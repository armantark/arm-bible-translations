# Product Context

## Why This Exists
Replaces DOCX-based workflow with an in-browser translation editor. DOCX files are no longer the source of truth â€” the JSON data files and web editor are.

## Problems Solved
- Manual copy/paste between Word and web is eliminated.
- Synchronized verse alignment across languages is handled automatically.
- Translation work happens directly in the browser with instant feedback.

## User Experience Goals
- Dark theme with clear, high-contrast reading interface.
- Noto Serif Armenian font for proper Armenian text rendering.
- Fast chapter/book switching via sidebars.
- Click on any verse to edit it; auto-saves on blur.
- Column visibility toggles (Armenian, English, Classical Armenian).

## Primary User
Translator/editor maintaining Armenian Bible translations, working in-browser and optionally exporting to Word for print.
